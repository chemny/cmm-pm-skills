① 屏幕流（屏与屏怎么跳）

flowchart TD
  A[首次启动] --> B[Onboarding 权限/快捷键/导入风格]
  B --> C[待命 后台]
  C -->|短按快捷键| D[录音浮窗 录音中]
  D -->|再短按| E[处理中]
  E --> F{三态判定}
  F -->|短/Shift| G[插入·原话]
  F -->|较长| H[插入·已润色]
  F -->|有指令| I[插入·已生成]
  G --> C
  H --> C
  I --> C
  C -->|打开设置| J[设置面板]
  D -->|无权限| K[错误·去授权]
  E -->|失败| L[错误·重试]
  C -->|额度用尽| M[升级/限额]

② 状态机（一次语音输入的状态流转）

stateDiagram-v2
  [*] --> 待命
  待命 --> 录音中: 短按快捷键
  录音中 --> 处理中: 再短按
  录音中 --> 待命: 取消(Esc)
  处理中 --> 插入: 判定(原话/润色/生成)
  处理中 --> 错误: 失败
  插入 --> 待命: 完成(可Ctrl+Z还原原话)
  错误 --> 待命: 重试/取消
